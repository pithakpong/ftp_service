และโครงสร้างหลักของโปรแกรมสามารถทำงานแบบ Read-Evaluate-Print-Loop (REPL) และรับคำสั่งได้

ทั้งนี้มีส่วนที่ไม่เหมือนต้นฉบับอย่างเดียวที่ผู้เรียนสังเกตได้คือ ข้อมูลการ tranfer ข้อมูล bytes/sec (เช่นคำสั่ง ls)
นอกเหนือจากนั้นผู้เรียนก็มั่นใจว่ามีการทำงานคล้ายคลึงกับ ftp client ของ microsoft ต้นฉบับ
